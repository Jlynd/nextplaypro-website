# NextPlayPro Website

A modern, responsive website for the NextPlayPro sports performance tracking application.

## Overview

NextPlayPro is a comprehensive sports performance tracking application designed for athletes, coaches, and teams. This website showcases the features and capabilities of the NextPlayPro platform.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and intuitive interface with smooth animations
- **Performance Analytics**: Track and visualize sports performance data
- **Goal Setting**: Set and monitor personal or team goals
- **Team Management**: Coordinate teams, schedules, and practices
- **Contact Form**: Easy way for visitors to get in touch

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Optional: Node.js and npm for running a local development server

### Running the Website

#### Option 1: Open Directly
Simply open `index.html` in your web browser.

#### Option 2: Using a Local Server
For the best development experience, use a local server:

```bash
# Using npx (no installation required)
npm start

# Or using Python
python -m http.server 8080

# Or using PHP
php -S localhost:8080
```

Then navigate to `http://localhost:8080` in your browser.

## Development

### Making Changes

1. Edit the HTML structure in `index.html`
2. Modify styles in `css/styles.css`
3. Update JavaScript functionality in `js/main.js`
4. Refresh your browser to see changes

### Adding Images

Place image files in the `images/` directory and reference them in your HTML:

```html
<img src="images/your-image.jpg" alt="Description">
```
